# Эксперименты: однослойный перцептрон

## 1. Влияние скорости обучения (learning rate)

**Параметры:** epochs=100, batch_size=32, инициализация — маленькие случайные веса.  
**Варианты:** η = 0.001, 0.01, 0.5, 1.0

### Наблюдения

| η      | Финальная accuracy | Сходимость          |
|--------|--------------------|---------------------|
| 0.001  | 0.9216             | Медленная, не сошлась полностью |
| 0.01   | 0.9412             | Плавная, стабильная |
| 0.5    | 0.9412             | Быстрая, лёгкие осцилляции |
| 1.0    | 0.9412             | Осцилляции, нестабильна вблизи минимума |

### Вывод

При малом η модель не успевает сойтись за фиксированное число эпох — шаг
слишком маленький (иначе будет сходиться очень долго). При большом η шаг превышает "ширину" минимума — модель
перескакивает через него и осциллирует. Оптимальный η выбирается под задачу;
для данного датасета значения 0.01–0.5 дают сопоставимое итоговое качество,
но с разной скоростью сходимости.

## 2. Влияние размера батча (batch size)

**Параметры:** epochs=100, lr=0.1, инициализация — маленькие случайные веса.  
**Варианты:** batch_size = 1, 16, 64, 256

### Наблюдения

| batch_size | Время обучения | Сходимость               |
|------------|----------------|--------------------------|
| 1          | 0.934s         | Шумная, сильные осцилляции |
| 16         | 0.066s         | Умеренный шум            |
| 64         | 0.022s         | Гладкая                  |
| 256        | 0.011s         | Очень гладкая, медленная в начале |

### Вывод

При batch_size=1 (SGD) градиент вычисляется по одному примеру — это
несмещённая, но высокодисперсная оценка истинного градиента, отсюда шум
на кривой loss. Также отсутствует параллельность вычислений → время
на эпоху максимальное. При большом батче градиент точный и стабильный,
но за эпоху происходит меньше обновлений весов. Оптимум — средние значения
(16–64): баланс скорости и стабильности.
PS: в таблице не приведен accuracy, так как при epochs=100 сходятся все варианты.

## 3. Влияние инициализации весов

**Параметры:** epochs=100, lr=0.1, batch_size=32.  
**Варианты:** нулевая, маленькие случайные (×0.01), большие случайные (×10)

### Наблюдения

| Инициализация | Финальная accuracy | Особенность графика        |
|---------------|--------------------|----------------------------|
| zeros         | 0.9412             | Нормальная сходимость      |
| small (×0.01) | 0.9412             | Нормальная сходимость      |
| large (×10)   | 0.9412             | Плато в первых эпохах      |

### Вывод

Большие начальные веса насыщают sigmoid: при больших значениях z функция
σ(z) выходит на плато (~0 или ~1), производная ≈ 0, градиент затухает —
модель "замирает" на старте (vanishing gradient из-за насыщения). Нулевая
инициализация не сломала однослойный перцептрон: здесь только один
выходной нейрон, симметрия не возникает. В многослойной сети нулевые
веса сделали бы все нейроны одного слоя идентичными, и они обучались
бы одинаково.
